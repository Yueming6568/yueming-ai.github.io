# 增加成员
在people文件夹下新开一个md文件，随意命名（建议以新同学的姓名拼音命名）。内容参考people\liken.md或以下模板：

```
name: 姓名
title: 博士生/博后/硕士生/研究助理/实验室行政人员/访问学生
period: 入学时间~结束时间
photo: 填照片的相对路径，照片文件放在./assets/img/people/下，建议256x256分辨率，图床也可以自动调整
homepage: 个人网页网址
google_scholar: 谷歌学术
email: 邮箱
github: Github链接
co_supervised: 可选
order: 放置在lab members页面中的顺序，从1开始
```

# PRLab 实验室主页重构 —— 预期效果说明

> 本文档描述重构完成后的视觉效果与用户体验，供设计/开发参考。

---

## 一、整体视觉印象

重构后的实验室主页将呈现**专业、干净、学术感强**的现代风格：

- **色调**：以深海军蓝（`#1B3A5C`）为主色，中蓝（`#2E86AB`）为强调色，去掉原有橙色；页面背景以白色和极浅灰蓝（`#F7F9FC`）为主，交替呈现各 Section，视觉层次清晰
- **字体**：延用无衬线系统字体栈，标题字重厚重（700/800），正文字重适中（400），行高宽松（1.7），易于长文阅读
- **排版**：大量留白，内容区最大宽度约 `1100px`，居中对齐，不显拥挤
- **参考气质**：类似 MMLab@NTU 的干净感 + zhanglinfeng.tech 的信息完整度

## 三、交互动效预期

| 交互 | 效果描述 |
|------|----------|
| 页面加载 | 各 Section 内容从下方淡入（`fadeInUp` 动画，可选） |
| 导航 ScrollSpy | 滚动时当前 Section 对应链接自动变蓝、出现下划线 |
| 卡片 Hover | 白色卡片上浮 `4px`，阴影加深，过渡 `0.25s ease` |
| 按钮 Hover | 描边按钮变为填充蓝色背景，白色文字，过渡 `0.2s` |
| 论文年份折叠 | 点击展开/收起，`max-height` 动画过渡，`0.3s ease` |
| New 展开折叠 | "View All News" 按钮展开剩余条目，箭头旋转 |
| 回到顶部按钮 | 蓝色圆形按钮，滚动超过 `300px` 后淡入，点击平滑滚回顶部 |
| 平滑滚动 | 所有锚点跳转使用 `scroll-behavior: smooth` |

## 四、移动端适配预期

| 元素 | 移动端表现 |
|------|-----------|
| 导航栏 | 汉堡按钮折叠菜单，点击展开 |

## 五、重构前后对比
